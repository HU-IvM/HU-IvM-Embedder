=== HU-IvM Embedder ===
Requires at least: 4.0
Tested up to: 5.9.3
Requires PHP: 7.0
Stable tag: 1.4.4

Plugin voor het embedden van diverse externe media op onze eigen WordPress installaties, geschreven voor HU-IvM.
